# chrome插件安装方法。

## 下载

- 下载插件代码到目录，比如c:/weixin.sidebar
- 或者：下载插件文件crx再解压到目录，比如weixin.sidebar.crx
- 下载地址github: https://github.com/qingmvc/weixin.sidebar

## 安装

- 进入chrome插件页面：chrome://extensions/
- 右上角打开“**开发者模式**”
- 选择按钮：“**加载已解压的扩展程序**”
- 选择插件目录：比如c:/weixin.sidebar
- 打开插件
- 右上角开启“**固定侧边栏**”
- 刷新页面：https://mp.weixin.qq.com/
